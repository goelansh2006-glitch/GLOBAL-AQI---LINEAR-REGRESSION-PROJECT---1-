# GLOBAL-AQI---LINEAR-REGRESSION-PROJECT---1-
This project applies Simple Linear Regression on a global air quality dataset to study the relationship between pollutants and AQI. It includes data preprocessing, model training, prediction, and visualization using matplotlib, helping analyze how pollution levels impact overall air quality.

This project focuses on analyzing global air quality data using Simple Linear Regression to understand the relationship between air pollutants and the Air Quality Index (AQI). The dataset, sourced from Kaggle, contains hourly pollution data from multiple cities and includes parameters such as PM2.5, PM10, NO2, SO2, CO, O3, and AQI. The project begins with data loading and preprocessing, where irrelevant data is removed and missing values are handled by selecting only numerical columns. A single pollutant feature is then used to train a linear regression model to predict AQI, making it a simple yet effective approach to understanding pollution trends.

The model is trained using a train-test split approach and evaluated using the R² score, which provides a measure of accuracy and how well the model explains the variation in AQI. Visualization is performed using matplotlib, where a scatter plot of the actual data points and a regression line helps in clearly understanding the relationship between the selected pollutant and AQI. This visual representation makes it easier to interpret how changes in pollutant concentration impact air quality levels.

Overall, this project demonstrates the practical application of machine learning techniques on real-world environmental data. It serves as a beginner-friendly implementation of regression analysis while also providing meaningful insights into air pollution patterns. The project can be further extended by applying multiple linear regression using multiple pollutants, experimenting with polynomial regression for improved accuracy, and comparing different models to achieve better predictions.
